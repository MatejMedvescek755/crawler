## Running the Application

1. Clone the repository to your local machine:
   ```bash
   git clone <repository-url>
   ```
2. Create a .env file in the project root and add your Etherscan API key:
   ```
   VITE_API_TOKEN=your-etherscan-api-key
   ```
3. Install the project dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm run dev
   ```
5. Open your browser and navigate to to view the application.
